Set gnome menus bar background color?
  Switch up from white to black on GCI Network Services, OSS covellite debian jessie
  8.00 amd64 workstation.  Did procedure on top menu bar and bottom bar on large
  left display on April 28, 2015.  Yes the display with menu bar switched from
  far right to far left.
 
  hold Alt-key right-click bar > left-click properties > Background
  x Solid color
  select black
  
  For tranparent drag selection all the way to the left.
  click Close


Gnome Shell Integration
=======================
Install Gnome Shell Integration in firefox (iceweasel) for configuration of 
Activities Configurator GNOME shell extension.

This plugin provides integration with Gnome Shell for extension enabling
and disabling.  It can be used only by extensions.gnome.org

Last Updated: 01/03/2015
File:         libgnome-shell-browser-plugin.so
MIME Types:   application/x-gnome-shell-integration \
              (Gnome Shell Integration Dummy Content-Type)

In firefox 
  triple lines > Add-ons > Plugins > Gnome Shell Intregration

  click Ask to Activate
  select Always Activate 
         Never Activate    <=== Unselected

Activities Configurator by nls1729[1]
=======================
Configure the Activities Button and Panel appearance with the Activities
Configurator Extension and Gnome Shell Extension Preferences Tool.

The gnome-shell-extensions package is installed already.
  gnome-shell-extensions - Extensions to extend functionality of GNOME Shell

Click the icon or text with the secondary mouse button to launch the Gnome Shell
Extension Preferences Tool:

  gnome-shell-extension-prefs
