GNOME Panel is a highly configurable launcher and taskbar for GNOME.
It formed a core part of the GNOME desktop in versions 1.x and 2.x.
It has been replaced in GNOME 3.x by default with GNOME Shell,
which only works with the Mutter window manager.
--en.wikipedia.org/wiki/GNOME_Panel


Syntax 
======
o Move icon:
Alt > right-click icon > Move 

o Remove icon:
Alt > right-click icon > Remove From Panel
